SoulArt Doodle Room — Colouring Section (v1)

1️⃣ Purpose (clear intent)

The Colouring Section is for:
	•	gentle nervous system regulation
	•	non-decision creativity
	•	presence without pressure

It is distinct from:
	•	Free Doodle (expression)
	•	Mandala Symmetry (flow + structure)

Colouring = receive, soften, settle

⸻

2️⃣ User Experience Flow

Entry
	•	User selects “Colouring Pages” mode
	•	Canvas clears
	•	A soft outline appears (no symmetry lines)
	•	Brushes default to soft brush, medium size, warm colour

User action
	•	Tap / click to colour inside shapes
	•	No pressure for precision
	•	Undo always available

Exit
	•	Switch back to other modes without losing saved state (same as now)

5️⃣ How Colouring Pages Work (important)

✅ Vector-style outlines rendered to canvas

❌ NOT flood-fill (too fiddly on mobile)

We will:
	•	Draw thick, soft outlines
	•	User colours freely inside
	•	Lines act as gentle guides, not strict borders

This avoids frustration and keeps it meditative.

7️⃣ Colouring Page Generator (Canvas-based)

Add this function:
Js:
function renderColouringPage(type) {
  ctx.clearRect(0, 0, canvas.width, canvas.height);

  const cx = canvas.width / 2;
  const cy = canvas.height / 2;
  const size = Math.min(canvas.width, canvas.height) * 0.4;

  ctx.save();
  ctx.strokeStyle = '#C8C3B8'; // soft warm stone
  ctx.lineWidth = 2.5;
  ctx.globalAlpha = 0.9;

  switch (type) {
    case 'lotus':
      for (let i = 0; i < 8; i++) {
        const angle = (i / 8) * Math.PI * 2;
        ctx.beginPath();
        ctx.ellipse(
          cx + Math.cos(angle) * size * 0.3,
          cy + Math.sin(angle) * size * 0.3,
          size * 0.25,
          size * 0.45,
          angle,
          0,
          Math.PI * 2
        );
        ctx.stroke();
      }
      break;

    case 'flower_of_life':
      for (let i = 0; i < 6; i++) {
        const angle = (i / 6) * Math.PI * 2;
        ctx.beginPath();
        ctx.arc(
          cx + Math.cos(angle) * size * 0.35,
          cy + Math.sin(angle) * size * 0.35,
          size * 0.35,
          0,
          Math.PI * 2
        );
        ctx.stroke();
      }
      ctx.beginPath();
      ctx.arc(cx, cy, size * 0.35, 0, Math.PI * 2);
      ctx.stroke();
      break;
  }

  ctx.restore();
  saveState();
}

Hook into selector
js:
document.getElementById('colouringSelect').addEventListener('change', function () {
  if (!this.value) return;
  currentColouringPage = this.value;
  renderColouringPage(this.value);
});

8️⃣ Brush Behaviour in Colouring Mode

Defaults when entering Colouring:
	•	Brush: Soft Brush
	•	Opacity: 80–90%
	•	Size: 10–14px
	•	Colour: warm neutral (sand / gold)

No eraser by default (still available).

This keeps it soothing.
